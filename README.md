# SnakeWaterGun Game

**SnakeWaterGun** is a simple yet engaging text-based game implemented in Python. It is a modern twist on the classic game "Rock-Paper-Scissors," adding a unique layer of strategy and fun.

## Game Description

In **SnakeWaterGun**, players choose one of three options: Snake, Water, or Gun. The rules are straightforward:

- **Snake** beats **Water** (Snake drinks the Water).
- **Water** beats **Gun** (Water extinguishes the Gun).
- **Gun** beats **Snake** (Gun shoots the Snake).

Each round, both the player and the computer make their choices, and the winner is determined based on the rules mentioned above. The game continues for a set number of rounds or until the player decides to quit.

## How to Play

1. Run the game script in your Python environment.
2. You will be prompted to enter your choice: Snake, Water, or Gun.
3. The computer will randomly select its choice.
4. The winner of the round will be announced, and the game will keep track of scores.
5. Play as many rounds as you like, or quit the game anytime.

## Features

- **Interactive Gameplay:** Enjoy a turn-based system with immediate feedback on the winner of each round.
- **Simple Rules:** Easy to understand and quick to pick up.
- **Random Computer Choices:** Each game is different with the computer making random selections.
